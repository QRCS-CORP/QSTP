\doxysubsection{Detailed Description}
The QSTP client state structure. 

This structure holds the state information for a client engaged in a key exchange. It contains\+:


\begin{DoxyItemize}
\item {\ttfamily schash\+:} The session token hash used to verify the integrity of the key exchange.
\item {\ttfamily serial\+:} The server certificate serial number, used to authenticate the server.
\item {\ttfamily verkey\+:} The server\textquotesingle{}s asymmetric signature verification key.
\item {\ttfamily expiration\+:} The expiration time (in seconds from the epoch) that indicates the validity period of the key exchange. 
\end{DoxyItemize}
