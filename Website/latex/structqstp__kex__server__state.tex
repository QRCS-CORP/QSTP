\doxysubsection{Detailed Description}
The QSTP server state structure. 

This structure holds the state information for a server participating in a key exchange. It includes\+:


\begin{DoxyItemize}
\item {\ttfamily prikey\+:} A pointer to the server\textquotesingle{}s private key for the asymmetric cipher.
\item {\ttfamily pubkey\+:} A pointer to the server\textquotesingle{}s public key for the asymmetric cipher.
\item {\ttfamily serial\+:} The certificate serial number acting as the key identity.
\item {\ttfamily schash\+:} The session token hash used to secure the key exchange.
\item {\ttfamily sigkey\+:} The server\textquotesingle{}s private signing key used for signing key exchange data.
\item {\ttfamily verkey\+:} The local asymmetric signature verification key.
\item {\ttfamily expiration\+:} The expiration time (in seconds from the epoch) for the key exchange session. 
\end{DoxyItemize}
